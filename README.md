# Mini LinkedIn (MERN, JS + JSX)

Features:
- Register/Login with JWT
- Profile (name, email, bio)
- Public text-only posts feed with author & timestamp
- Profile page shows user's details & posts
  

# Stack Used
   - Frontend- React
   - Backend- Node.js
   - Database- Mongodb
## Quick Start

### 1) Backend
```bash
cd backend
cp .env.example .env   
npm install
npm run dev            
```
API runs on `http://localhost:5000`.

### 2) Frontend
```bash
cd ../frontend
cp .env.example .env   
npm install
npm run dev        
```
